== POKÉDEX on React.js

{Pokédex on React}[https://pokedex-on-react.herokuapp.com/] is a Ruby on Rails applicaiton paired with React.js. It was constructed to understand the mechanics of the {FlipMove}[https://github.com/joshwcomeau/react-flip-move] component library and apply it to a single page application. This application also scrapes {pokemonqrcode.com}[http://pokemonqrcode.com/] and {pokemondb.net}[http://pokemondb.net/pokedex/all] to gather the data on the original 151 Pokémon. {Pokédex on React}[https://pokedex-on-react.herokuapp.com/] displays the functionality of deleting, searching and shuffling Pokémon in a single page application.
